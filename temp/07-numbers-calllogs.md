# Voqo AI - Numbers & Call Logs

## Numbers Management

### Overview
Numbers feature enables purchasing and managing multiple phone numbers directly within Voqo AI platform.

### Purchasing Numbers

**Available Regions**
- Australia (AU)
- United States (US)

Numbers acquired from downstream suppliers through the platform.

### Number Assignment

**Assigned Numbers**
- Route calls directly to specific agents
- One-to-one mapping for dedicated lines

**Unassigned Numbers**
- Flexible, shared use across agents
- Not tied to any specific agent

### Management Operations

Access via: `https://platform.voqo.ai/numbers` or sidebar "Numbers" section

**Operations Available**
| Operation | Description |
|-----------|-------------|
| Add | Purchase new numbers from available regions |
| Update | Modify existing number details |
| Reassign | Move number to different agent |
| Delete | Remove number when no longer needed |

Dashboard displays all numbers associated with account.

### Pricing

**Default Number (Complimentary)**
- One phone number included free with paid subscription
- No additional cost for base tier users

**Additional Numbers**
- Monthly recurring fee per extra number
- Beyond the complimentary default number

## Call Logs

### Overview
Centralized call history with chronological ordering. Access via platform call logs section or sidebar.

### Data Stored Per Call

| Field | Description |
|-------|-------------|
| Call Summary | Brief overview of conversation |
| Caller Number | Phone number involved in call |
| Duration | Length of call |
| Recording | Audio playback (if enabled) |
| Time Zone | Agent's timezone (editable in agent settings) |
| Transcript | Full call text with function calls + timestamps |

### Call Details View

Selecting any call entry opens detailed panel containing:
- Complete conversation transcript
- System function call documentation
- Precise timestamps throughout
- Post-call action results

### Post-Call Actions

Automatic task execution after call completion:
- Results accessible within call log entry
- Shows what actions were triggered
- Displays action outcomes

### Recording Playback

**Availability**
- Requires recording enabled in agent's advanced settings
- Audio files accessible when feature is on

**Retention Period**
- 6 months storage for recordings

### Transcript Details

**Contents Include**
- Full conversation text
- System function calls documented
- Timestamps for each segment

**Generation Note**
- May require a few minutes after call concludes
- Not instant availability

### Use Cases

- Performance auditing
- Compliance record-keeping
- Agent training
- Dispute resolution
- Call content verification
